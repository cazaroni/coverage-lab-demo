# Inference Service (Phase 0 Scaffold)

This directory contains the Inference Phase 0 bootstrap only.

Current scope:
- BentoML-friendly Python service layout.
- `/healthz` and readiness endpoint semantics.
- Bundle/manifest validation placeholders.
- Observability hook placeholders.

Out of scope for Phase 0 scaffold:
- Real model loading.
- Real scoring or batch inference.
- Any play lookup or `play_id` resolution.
