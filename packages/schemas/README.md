# ProjectEdge Schemas (Phase 0)

This package provides lean, shared data contracts for the Data workstream Phase 0 foundation.

Included in this scaffold:

- Canonical lake prefix templates
- Stable join-key definitions
- Schema registry placeholders for required dataset schemas
- Pydantic manifest models for dataset metadata
- Sample fixture and manifest placeholders for downstream teams

This package intentionally does **not** implement full ingest validation or orchestration logic in Phase 0.
